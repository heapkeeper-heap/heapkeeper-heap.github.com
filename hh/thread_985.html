Gravatar profile pictures in HTML pages

Csaba Hoch Gravatar profile pictures in HTML pages [idea, prop] <hh/985> (2009-12-06) Children: - 
[effort 4]

We could have profile pictures in the generated HTML pages. The
easiest way would be to use Gravatar pictures.

It would be nice if the user would not have to be online to be able to
use the Gratavar pictures. So I propose a command that
downloads/updates Gravatar pictures of the authors and stores them
locally, and the generator commands should just use those locally
stored image files.
