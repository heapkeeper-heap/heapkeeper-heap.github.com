When to use tag and when meta text

Csaba Hoch When to use tag and when meta text [hh-conventions, reviewed] <hh/21> (2009-03-19) Children: - 
The advantage of using a tag over a piece of meta text is that it is
easy to see how the tags are inherited.

A rule of thumb: if the usual usage for a meta-thing is
to be inherited by the posts' children, then it should be a tag;
otherwise, it should be a metatext command.

An application of this rule: "delthis" should be a tag and not a meta
command when we start talking about something that should not be kept.
