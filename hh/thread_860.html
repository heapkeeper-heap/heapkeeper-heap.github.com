If I think right, the way to add sections in `hkgen.Generator` is to
subclass and override `print_main_index_page`. Am I right?
Csaba Hoch — <hh/861> (2009-11-02) Parent: hh/860 Children: - 
It depends. If don't want the "default" index page to be generated,
then yes. If you want a new page (e.g. attis_index.html) beside the
default index, then you can write print_attis_page+write_attis_page
methods and modify the `write_all` method so that it will print that
as well.
