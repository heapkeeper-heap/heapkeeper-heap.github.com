Deprecating hkgen.Generator.{print_post_page,write_post_pages}

Csaba Hoch Deprecating hkgen.Generator.{print_post_page,write_post_pages} [issue, removal] <hh/1382> (2010-09-19) Children: - 
The print_post_page and write_post_pages methods of hkgen.Generator are
used only for generating post pages. In a previous thread [1] we already
agreed that post pages are unnecessary and we use thread pages
instead. So I propose making them deprecated in v0.8 and removing them
from v0.9.

If you agree, it will also mean that we can close heap://hh/749 which
talks about a possible modification to post page generation.
