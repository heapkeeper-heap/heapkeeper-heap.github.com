[effort 5]

It is getting harder to keep track of all the tags we use.

We could store all accepted tags in a file in the heap's directory
(e.g. "known_tags"), and the Generator could highlight the tags that
are not present in this file. Then the maintainer could decide either
to correct the tag (e.g. correct "Python" to "python"), or to add the
tag to the known_tags file.
